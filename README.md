SETUP
-
MIGRATIONS
1. Run Migrations
2. Open Package Manager Console
3. Set SabioExam.DATA as default project
4. Run `Update-Database`
-
APP START
5. Right click solution > Properties
6. Set `Action = Start` for SabioExam.CLIENT and SabioExam.SERVER 
7. Apply / Save
-
RUN PROJECT
